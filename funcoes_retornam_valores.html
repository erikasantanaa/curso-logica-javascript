Pedir para o usuário digitar dois números que representam os lados de um retângulo. Mostrar na tela a área total de duas formas distintas: Num alert ou na div#output (dependendo do botão cliclado). Esse código não precisa ser executado quando a página é aberta.

Funções que retornam valores são úteis para que o desenvolvedor tenha autonomia para decidir o que deseja fazer com o valor retornado.
